Dicoding Story is an Android apps that uses a Private API from Dicoding with user login authentication where users can upload a stories (with pictures and text) that can be seen also by other users. In addition, this application also has a Google Maps API that users can use to add stories along with their current location. Each user can also see the created story along with its location.
