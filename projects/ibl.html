IBL Teams

Project Information

  • Category : Android (Kotlin)
  • Project Date : Mar 2021 - Jul 2021
  • Find more on :  

IBL Teams is a simple Android apps that implement RecyclerView and using static data that I collected from the official website. The idea is that users can get an overview about Indonesian's basketball teams through offline mode or without using any internet to access. This project is for completing the submission in the Belajar Membuat Aplikasi Android untuk Pemula course at Dicoding Indonesia which became my first-time project in building an Android apps using Kotlin. I made this project from scratch until it can be used by myself and I got 5/5 stars for this submssion rating.
