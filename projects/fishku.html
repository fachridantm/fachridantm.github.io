Fishku is a startup in the fisheries e-commerce sector that aims to improve the economy sector in Indonesia. Not only for fish transactions, users can also detect fish freshness by using machine learning model through an android apps. Trained machine learning model will be deploy in Google Cloud Platform as our server services. Then, the deployed model will be store into databases so it can be used in android apps by using API transactions.
